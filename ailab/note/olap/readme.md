
# 简单APP分析指标

## 指标说明

- 正确区分指标和维度
- 指标:可以度量的值,例如:True(男),23岁,175CM,85KG.
- 维度:数据的属性,例如:性别,年龄,身高,体重.
- 粗放式增长[蛮荒时代]VS精细化运营[群魔乱舞]
- 同质化VS精细化
